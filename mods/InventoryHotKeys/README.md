Adds some shortcut in the inventory screen, when hovering over an Item.
---
Shortcuts are
* Consume Item - Shift + E
* Remove Mag. from Weapon - Shift + R
* Unload Mag. - Shift + R
* Split stacked Item - Shift + S
* Take X from stacked Item - Alt + S
* Remove Laser from Weapon - Shift + T
* Remove Optic sight from Weapon - Shift + F
* Remove Muzzle from Weapon - Alt + F
---
Note: Currently there is no plan to add MCM configuration for custom key bindings. Sorry :(
